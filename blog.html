What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page

How do you select an element from the DOM?

The getElementById() method returns an element with a specified value. The getElementById() method returns null if the element does not exist. The getElementById() method is one of the most common methods in the HTML DOM. It is used almost every time you want to read or edit an HTML element.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a pattern in the context of the Document Object Model (DOM) where a single event listener is added to a parent element instead of individual event listeners to multiple child elements. This parent element listens for events triggered by any of its child elements, and when an event occurs, it "delegates" the handling of that event to the appropriate child element by using event bubbling (or capturing).

How do you manipulate an element's attributes and styles using the DOM?

Attributes: Use .getAttribute(), .setAttribute(), .removeAttribute(), and .hasAttribute() to manipulate element attributes. Styles: Use the .style property for inline styles or classList for managing CSS classes.
